# Simple Library Manager 📚

Ein Python-basiertes System zur Verwaltung von Büchern, entwickelt im Alter von 17 Jahren als Teil meiner Vorbereitung auf ein Informatikstudium.

## Features
* **Daten-Persistenz:** Nutzt JSON-Dateien, um Daten dauerhaft zu speichern.
* **CRUD-Funktionen:** Bücher können erstellt (Create), gelesen (Read) und gelöscht (Delete) werden.
* **Error Handling:** Robustes System mit `try/except`, das auch bei fehlenden Dateien nicht abstürzt.

## Technischer Stack
* Sprache: Python 3
* Datenformat: JSON

## Warum dieses Projekt?
Dieses Projekt zeigt mein Verständnis für den Übergang von flüchtigen Objekten im Arbeitsspeicher zu dauerhafter Datenspeicherung. Es war ein wichtiger Meilenstein beim Erlernen von Algorithmen und Datenstrukturen.
